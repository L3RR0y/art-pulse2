# 🎨 ART PULSE — Галерея известных художников

Интерактивная галерея, посвящённая великим художникам и их творчеству.

## 🌐 Сайт
https://l3rr0y.github.io/art-pulse2/

## 📖 О проекте
Проект представляет собой веб-галерею с информацией о 50+ известных художниках. 
Сайт позволяет знакомиться с биографиями, изучать известные произведения, 
сохранять любимых художников в избранное и проходить викторину для проверки знаний.

## 🛠️ Функциональность
- 🖼️ Галерея художников с карточками
- 🔍 Поиск по имени или стилю
- 🏷️ Фильтрация по художественным направлениям
- ❤️ Избранное (сохраняется в локальном хранилище)
- ⚖️ Сравнение художников
- 📚 Детальная информация о каждом художнике (биография, известные работы)
- 🖼️ Просмотр картин в полноэкранном режиме
- 🎯 Викторина для проверки знаний об искусстве
- 🌓 Светлая/тёмная тема
- 📱 Адаптивный дизайн для мобильных устройств
